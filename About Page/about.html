About Color Changer

This is a color changing website, On Clicking the any one box , the background color of the main section of the page gets change to the color of the box you clicked.

Introduction

This project is designed to change the background color of the page by clicking on color boxes.

How It Works

Click on any of the color boxes in the home page to change the background color.
